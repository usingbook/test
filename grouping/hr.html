웹문서의 작업

웹문서는 구조(HTML), 표현(CSS), 동작(Javascrippt)을 분리해서 작업을 해야합니다.

HTML 이란 Hyper Text Markup Language로 웹에서 사용되는 문서를 작성하는 마크업 언어입니다. HTML은 문단(paragraph), 목록(list), 표(table)등의 내용을 태그를 이용해 구조화합니다.

CSS 란 Cascading Style Sheets로 웹문서를 의도한 디자인에 맞게 표현될 수 있도록 하는 스타일시트입니다. CSS는 서체,색상, 배경, 문단모양 ,목록 등의 스타일을 지정하여 HTML을 보다 전달력있고 심미적으로 표현할 수 있습니다

Java Script란 (줄여서 JS라고 불리기도 합니다) 웹 문서에서 사용되는 스크립팅 언어입니다. JS는 컴파일러가 필요없는 인터프리터 언어로 스크립트 표준안이 ECMAScript 표준을 기준으로 합니다. JS는 HTML과 CSS를 동적으로 수정, 삭제, 추가 하며 마우스, 키보드, 창 등의 이벤트에 따른 동작을 가능하게 합니다.
